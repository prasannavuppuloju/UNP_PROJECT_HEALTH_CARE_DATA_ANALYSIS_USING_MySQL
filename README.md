# UNP_PROJECT_MEDIPLUS_DATA_ANALYSIS_USING_MySQL


I did this project in collaboration with UNP, where I created a Hospital Management System using SQL to understand and manage hospital operations efficiently. The database stores important details about patients, doctors, appointments, medicines, prescriptions, and billing in a structured way. It highlights key information such as total patients, total appointments, total revenue, and medicine stock. With the help of SQL queries like JOIN, GROUP BY, and aggregate functions, it is easy to analyze data, track patient–doctor interactions, monitor appointment status, and identify pending payments. The project also includes solving practical queries such as finding completed appointments, calculating revenue, and analyzing medicine usage. Overall, this project helps in organizing hospital data effectively and supports better decision-making through easy data retrieval and analysis.
