# UNP_PROJECT_HEALTH_CARE_DATA_ANALYSIS_USING_MySQL


I did this project in collaboration with UNP. This project is a Hospital Management System named Mediplus developed using SQL. The main aim of this project is to store and manage hospital data in a structured and organized manner. It includes tables such as Patients, Doctors, Appointments, Medicines, Prescriptions, Prescription_Details, and Billing. These tables are connected using foreign keys to maintain relationships between data.We used SQL queries like JOIN, GROUP BY, and aggregate functions to retrieve useful information such as patient details, doctor interactions, total revenue, and appointment status. This project helped us understand how databases are used in real-world hospital systems and improved our practical knowledge of SQL.
